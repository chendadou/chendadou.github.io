关于 Javascript 的 this 和闭包

Posted on 2017-05-20
  • this 是 Javascript 中的一个关键字；
  • 对闭包的理解：
    • 能够读取其他函数内部变量的函数；
    • 定义在一个函数内部的函数；
    • 本质上来说，闭包就是将函数内部和函数外部连接起来的一座桥梁。

闭包用途
